## 파밍파밍해

### 제품설명
- 월급, 일당, 시급을 입력 하면 초당 얼마를 벌고 있는지 실시간으로 표시
- 일하기 싫을때 이거라도 보면서 시간때우기 위함

### 구현 및 extension 구조
- javascript
- html

```
extension/
 ├─ manifest.json
 ├─ farming.png
 ├─ shared.js
 ├─ windows.js
 ├─ background.js
 └─ windows.html
```

### 설치방법
1. https://chromewebstore.google.com/ 에 접속
2. 파밍파밍해 검색
3. 다운로드 후 확장프로그램 설치
4. 항상 핀고정
5. 잘 사용
